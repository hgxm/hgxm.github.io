Array对象类型常用方法总结

2017-03-22

判断是否为数组
（1）typeof

(2) instanceof

(3) Array.isArray

Array实例的方法（按功能归类）：
（一）转换数组
（1）toString

➡️功能：返回数组的字符串形式
➡️参数：无
➡️返回值：字符串
➡️是否改变原数组：否

（2）valueof

➡️功能：返回数组本身
➡️参数：无
➡️返回值：数组
➡️是否改变原数组：否

（二）栈方法
（1）push()

➡️功能：在数组末端添加一个或多个元素
➡️参数：
➡️返回值：修改后数组长度
➡️是否改变原数组：是

（2）pop()

➡️功能：删除数组的最后一个元素
➡️参数：
➡️返回值：返回移除的元素
➡️是否改变原数组：是

（三）队列方法
（1）shift()

➡️功能：删除数组的第一个元素
➡️参数：无
➡️返回值：返回删除的元素
➡️是否改变原数组：是

(2)unshift()

➡️功能：在数组的第一个位置添加元素
➡️参数：无
➡️返回值：返回新数组的长度
➡️是否改变原数组：是

（四）排序方法
（1）reverse()

➡️功能：倒置数组中元素的位置
➡️参数：无
➡️返回值：新数组
➡️是否改变原数组：是

（2）sort()

➡️功能：按照字典顺序对数组成员进行排序
➡️参数：无
➡️返回值：新数组
➡️是否改变原数组：是

（五）操作方法
（1）concat()

➡️功能：用于多个数组的合并，将新数组添加到原数组的尾部
➡️参数：数组或其他数据类型
➡️返回值：新数组
➡️是否改变原数组：否

（2）slice()

➡️功能：提取（切割、切片）原数组的一部分
➡️参数：一个或两个参数。第一个参数：起始位置；第二个参数：终止位置（但不包括该位置）。（注意1：若参数中出现负数，则用数组长度加上该数来确定相应位置。注意2：如果结束位置小于开始位置，则返回空数组）
➡️返回值：新数组
➡️是否改变原数组：否

（2）splice()

➡️功能：最强大数组方法。可以用来删除、插入、替换数组元素
➡️参数：
· 删除：2个参数。要删除的起始位置和要删除的项数
· 插入：3个参数。起始位置、0、要插入的元素(一个或多个)
· 替换：3个参数。起始位置、要删除的元素、要插入的元素(一个或多个)
➡️返回值：
· 删除：返回被删除的元素
· 插入：空数组
· 替换：返回被删除的元素
（总结：splice方法总会返回一个数组，该数组包含从原始数组中删除的项，如果没有任何删除的项则返回一个空数组）
➡️是否改变原数组：是

（六）位置方法
（1）indexOf()

➡️功能：返回元素在数组中第一次出现的位置，没有找到则返回-1
➡️参数：一个或两个参数。要查找的元素、搜索的开始位置
➡️返回值：该元素出现的位置或者-1
➡️是否改变原数组：否

（2）lastIndexOf()

➡️功能：返回元素在数组中最后一次出现的位置，没有找到则返回-1, 该方法从数组末尾向前查找
➡️参数：一个或两个参数。要查找的元素、搜索的开始位置
➡️返回值：该元素出现的位置或者-1
➡️是否改变原数组：否

（七）迭代方法（数组的迭代方法都接受两个参数：要在每一项上运行的函数和该函数的作用域对象，而该函数会接受三个参数：数组项的值，该项在数组中的位置和数组对象本身）
（1）forEach()

➡️功能：循环数组并对数组中的每一项运行指定函数
➡️参数：
➡️返回值：无
➡️是否改变原数组：否

（2）map()

➡️功能：循环数组并对数组中的每一项运行指定函数
➡️参数：
➡️返回值：返回每次函数调用的结果返回的数组
➡️是否改变原数组：否

（3）filter()

➡️功能：循环数组并对数组中的每一项运行指定函数
➡️参数：
➡️返回值：返回筛选后结果和true的元素组成的新数组
➡️是否改变原数组：否

（4）every()

➡️功能：循环数组并对数组中的每一项运行指定函数。如果函数对每一项都返回true，则返回true
➡️参数：
➡️返回值：true/false
➡️是否改变原数组：否

（5）some()

➡️功能：循环数组并对数组中的每一项运行指定函数。如果函数对任一项都返回true，则返回true
➡️参数：
➡️返回值：true/false
➡️是否改变原数组：否

（1）reduce()

➡️功能：
➡️参数：
➡️返回值：
➡️是否改变原数组：

（2）reduceRight()

➡️功能：
➡️参数：
➡️返回值：
➡️是否改变原数组：
